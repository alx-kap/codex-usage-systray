# ChatGPT Codex Usage Tray v1.0.5

## Highlights

- Moved the menu experience onto macOS's native `MenuBarExtra` window presentation for a more system-like feel
- Fixed the clipped dropdown layout by letting the menu size itself naturally
- Restored the compact menu bar label so both 5 Hour and Weekly usage appear at a glance
- Tightened the panel width and refined spacing to better match Apple's built-in menu extras

## Visual Polish

- Improved translucency and overall "liquid glass" styling
- Darkened section dividers to better match native macOS menus
- Darkened row hover states so actions feel closer to built-in menu selection behavior
- Smoothed refresh status copy with clearer messages like `Refreshing usage...` and `Updated just now`

## Notes

- Release artifact: `ChatGPT-Codex-Usage-Tray-v1.0.5-macOS.zip`
- This build was verified with a successful macOS Release build in Xcode's command-line toolchain
